# Python Banking Project for Beginners

A simple and beginner-friendly banking system project built with Python. This project is designed for beginners who want to understand basic Python programming concepts such as variables, loops, conditional statements, and user input through a real-world banking simulation.

## About the Project

This Python banking project simulates basic banking operations in a simple command-line interface. It is built using procedural programming without functions or classes, making it easy to understand for beginners.

## Features

- Account creation
- Money deposit
- Money withdrawal
- Balance inquiry
- Exit system

## Built With

- Python 3

## Python Concepts Used

- Variables
- Input and Output
- While Loop
- If-Elif-Else Conditions
- Arithmetic Operations
- Program Flow Control

## Installation and Usage

### Clone the repository

```bash
git clone https://github.com/sehranrasooljan/Python_Banking_Project_Project_for-Beginners.git
```

### Go to the project directory

```bash
cd Python_Banking_Project_Project_for-Beginners
```

### Run the project

```bash
python bank.py
```

## Example Usage

1. Enter your name to create your bank account  
2. Select an option from the menu  
3. Deposit money into your account  
4. Withdraw money if balance is available  
5. Check your current balance  
6. Exit the banking system
